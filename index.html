Desarrollo E-Commerce "ONE TO ONE"

Proyecto integrador para Digital House

Fecha: Agosto 2023 - Abril 2024

En este proyecto, he desarrollado una aplicación e-commerce integral, ONE TO ONE, como parte de mi formación en Digital House. El proyecto cubrió desde el diseño inicial hasta la implementación de la funcionalidad completa, utilizando un stack tecnológico avanzado.

Tecnologías Aplicadas:

  • JavaScript: Para la lógica de la aplicación y la interacción del usuario.
  • Node.js: En el backend para manejar la lógica del servidor y las operaciones asíncronas.
  • Express: Framework para construir y gestionar el servidor de manera eficiente.
  • HTML & CSS: Para el diseño y maquetado del front-end, garantizando una interfaz atractiva y funcional.
  • MySQL (Sequelize): Para la gestión y manipulación de la base de datos relacional.
  • React: Biblioteca para construir interfaces de usuario dinámicas y responsivas.
  • REST: Para el diseño de la arquitectura de la API, facilitando la comunicación entre el frontend y el backend.
  • MVC: Patrón de diseño para separar las preocupaciones y estructurar la aplicación.
  • Git & GitHub: Para el control de versiones y colaboración en el código fuente.
  • Jira: Herramienta para la gestión de proyectos y seguimiento de tareas.

Este proyecto ha sido una experiencia valiosa que ha permitido aplicar y consolidar habilidades en desarrollo web, desde la creación de interfaces hasta la implementación de funcionalidades complejas.
